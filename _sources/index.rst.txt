Vicoco
======

**Vicoco** provides cocotb_ support for the Vivado_ `xsim` simulator. 

Developed and maintained by Kiran Vuksanaj, and originally built as a tool
for the Digital Systems Laboratory (6.205 / 6.111) course at MIT.

This project began as a fork of Tomasz Hemperek's cocotb_vivado_.

.. _Vivado:
   https://www.amd.com/en/products/software/adaptive-socs-and-fpgas/vivado.html
.. _cocotb:
   https://www.cocotb.org
.. _cocotb_vivado:
   https://www.github.com/themperek/cocotb-vivado
